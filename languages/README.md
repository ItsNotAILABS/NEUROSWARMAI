## Spec File Format

Every `.spec` file contains:
- **Purpose** — What the language does
- **Syntax** — How to write it
- **Semantics** — What it means
- **Examples** — Living code samples
- **Integration Points** — How it connects to other languages
- **Terminal Bindings** — How terminals use it
- **Atlas Bindings** — How Atlas governs it
