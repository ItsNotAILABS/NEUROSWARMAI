# JARVIS Extension Icons

Replace with JARVIS icon PNGs at 16x16, 32x32, 48x48, 128x128 sizes.

## Required files

- `jarvis-16.png` — 16×16 toolbar icon
- `jarvis-32.png` — 32×32 toolbar icon
- `jarvis-48.png` — 48×48 extensions page icon
- `jarvis-128.png` — 128×128 Chrome Web Store / install dialog icon

## Design guidelines

- Use dark background (#0a0a0b) with cyan (#22d3ee) accent
- "J" letterform or arc reactor motif
- Ensure legibility at 16×16
