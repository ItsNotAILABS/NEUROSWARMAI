## Relationship to Atlas

The Sovereign Terminal is NOT Atlas. Atlas is the meta-governor —
a system. The Sovereign Terminal is the **person** who governs the
system. Atlas reads ACL. The Sovereign Terminal writes ACL.

```
Sovereign Terminal
    │
    ├── Writes → ACL (Atlas configuration)
    ├── Enforces → CPL-L (through Atlas)
    ├── Evolves → UEL (the universe itself)
    │
    └── Atlas
        ├── Reads ACL
        ├── Enforces CPL-L
        ├── Manages Registry
        ├── Oversees Terminals
        └── Evolves via UEL
```
