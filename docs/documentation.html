🚀 Quick Start

Get up and running with CHIMERIA Defense in under 5 minutes

# Install the CHIMERIA CLI
$ npm install -g @chimeria/cli
# Authenticate with your API key
$ chimeria auth login --key YOUR_API_KEY
# Deploy your first defense configuration
$ chimeria deploy --config defense.yaml
# Check system status
$ chimeria status --all
✓ 10/10 defense agents ACTIVE
